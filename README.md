# 环境搭建:
- 安装Node.js环境
    [Node.js中文网](http://nodejs.cn/)
- 安装vue-cli 脚手架工具
    - `npm install -g vue-cli` 直接安装
    - 利用淘宝镜像 
        `npm install -g cnpm --registry=https://registry.npm.taobao.org`
        [淘宝NPM 镜像](http://npm.taobao.org/)
- 安装完成之后  
    `vue --version` 查看版本  
    `vue init webpack 项目名字`  
    `cd 项目名字`  
    `npm install`  
    ------complete------  
    `npm run dev`
